# React + Vite

🎮 Tres en Raya

Nombre: Hans Benny Hidalgo

Descripción del juego

Este proyecto consiste en el desarrollo del clásico juego Tres en Raya (Tic-Tac-Toe) utilizando React. El jugador compite contra una inteligencia artificial básica que realiza movimientos automáticos en el tablero. El juego detecta automáticamente el ganador o si ocurre un empate, y permite reiniciar la partida.

✍(◔◡◔) Qué aprendí
Uso de useState para manejar estados dinámicos
Uso de useEffect para guardar datos en el navegador con localStorage
Creación de componentes en React
Lógica para detectar condiciones de victoria
Manejo de eventos en botones

╰（‵□′）╯Qué parte fue más difícil

La parte más difícil fue implementar la lógica para detectar el ganador correctamente y lograr que la inteligencia artificial jugara de manera automática sin errores.
